## SpringMVC概述

​	SpringMVC是一种基于Java实现的MVC设计模型的请求驱动类型的轻量级web框架, Spring框架提供了构建web应用程序的全功能MVC模块, 使用Spring可插入的MVC架构, 从而在使用Spring进行WEB开发时, 可以选择使用Spring的SpringMVC框架或集成其他MVC开发框架.
